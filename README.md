# 2D Mini Web Game 🎮

A minimalist 2D web game built with native web technologies. It handles basic keyboard inputs, player movement, and collision logic within the browser viewport.

---

## 🚀 Features

* **Game Loop:** Basic JavaScript loops to update entity positions.
* **Input Handling:** Event listeners for keyboard keystrokes to move the player.
* **Collision Detection:** Simple boundary checks to keep the player inside the game area.

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript (ES6)
